# SoD
Simulation of DNA sequences using multiple context models

## Installation

<pre>
cd src /
make
</pre>

## Example

<pre>
./SoD -u 5 a=1/1 d=1/10 bs 50 -o out -n 200 ecK12DH10B.fa.gz
cat out
</pre>
